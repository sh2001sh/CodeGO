## What is shu26.cfd?

**shu26.cfd** is an AI relay and model access platform focused on coding workflows. It is designed for users who want to access models used by:

- Codex
- Claude Code
- Gemini CLI
- OpenAI-compatible tools
- AI coding clients and agent tools

The platform is intended for developers who care about availability, centralized account management, model selection, and easier tool integration.

## What is CodeGo?

**CodeGo** is the desktop control center for shu26.cfd. Instead of manually editing local config files for every tool, CodeGo gives you one desktop app to connect your account, manage API configuration, inspect keys, review logs, and sync model-related information with the website.

CodeGo is especially useful when you need to:

- switch between Codex, Claude Code, and Gemini CLI more efficiently
- use a relay endpoint without repeatedly editing JSON, TOML, or environment variables
- manage keys and models from one place
- inspect logs and account status from the desktop
- keep local tool configuration aligned with the website account

## Why developers use CodeGo

- **Built for AI coding workflows**: CodeGo is focused on Codex relay, Claude Code relay, Gemini CLI relay, and similar developer use cases.
- **Desktop management instead of manual edits**: configure supported tools through a visual app instead of editing raw files by hand.
- **Website + desktop workflow**: shu26.cfd handles account, keys, models, and platform pages; CodeGo brings that workflow to a local desktop app.
- **Cross-platform desktop client**: available for Windows, macOS, and Linux.

## Core use cases

### Codex relay desktop workflow

Use shu26.cfd as your Codex relay platform, then use CodeGo to manage the local configuration, account connection, keys, and model selection workflow on your machine.

### Claude Code relay desktop workflow

If you use Claude Code with a relay endpoint, CodeGo helps you manage account-linked API configuration and reduces repeated manual switching.

### Gemini CLI relay desktop workflow

For Gemini CLI users, CodeGo provides a more centralized way to manage relay-related settings and platform-linked configuration.
