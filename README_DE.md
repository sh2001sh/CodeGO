# CodeGo

CodeGo ist der Desktop-Client für **[shu26.cfd](https://shu26.cfd)**. Die Anwendung richtet sich an Entwickler, die **Codex**, **Claude Code**, **Gemini CLI** und ähnliche AI-Coding-Tools mit einer Relay- oder Gateway-Umgebung nutzen.

## Was ist shu26.cfd?

**shu26.cfd** ist eine Plattform für AI-Relay- und Modellzugriff mit Fokus auf Coding-Workflows. Relevante Einsatzfälle sind unter anderem:

- Codex relay
- Claude Code relay
- Gemini CLI relay
- OpenAI-kompatible API-Workflows

## Was ist CodeGo?

**CodeGo** ist die Desktop-Steuerzentrale für shu26.cfd. Statt mehrere lokale Konfigurationsdateien manuell zu pflegen, verwaltest du Kontoanbindung, API-Konfiguration, Schlüssel, Modelle und Logs in einer einzigen Desktop-App.

## Wofür wird CodeGo genutzt?

- Verbindung mit dem shu26.cfd-Konto
- Verwaltung lokaler API-Konfigurationen
- Auswahl und Anwendung verfügbarer Keys
- Anzeige von Modelllisten
- Zugriff auf Model-Plaza-Informationen
- Anzeige des Gruppenstatus
- Einsicht in Logs
