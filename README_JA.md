CodeGo は **[shu26.cfd](https://shu26.cfd)** のデスクトップクライアントです。**Codex**、**Claude Code**、**Gemini CLI** などの AI コーディングツールを使う開発者向けに、より整理されたローカル設定と中継利用フローを提供します。

## shu26.cfd について

**shu26.cfd** は、AI コーディング用途に特化したモデル接続・中継プラットフォームです。主な想定シナリオ:

- Codex relay
- Claude Code relay
- Gemini CLI relay
- OpenAI 互換 API ワークフロー

## CodeGo について

**CodeGo** は shu26.cfd のデスクトップ管理アプリです。複数のツールごとに設定ファイルを手動編集する代わりに、1 つのアプリでアカウント接続、API 設定、キー選択、モデル確認、ログ確認を行えます。

## 主な用途

- shu26.cfd アカウントとの接続
- デスクトップ側 API 設定の管理
- 利用可能キーの選択と適用
- モデル一覧の確認
- モデル広場ページの表示
- グループ状態ページの表示
- ログ確認
